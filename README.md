# Kia Takat큰 훮-Matihiko - Content Pipeline Assistant

This website has been developed to assist authors to efficiently create and develop content for [Kia Takat큰 훮-Matihiko](http://www.kiatakatu.ac.nz/), the National Digital Readiness programme.

Goals of website:

- Track content as it is developed, from draft to release.
- Allow authors to see the links between content, and travel through these links.
- Allow authors to see coverage of content against specific goals.
